# 🎨 Proyecto Aula Dinámica - Juegos Bíblicos (Asambleas de Dios)

Este es un sistema interactivo web desarrollado en **PHP** diseñado para niños. Incluye tres fases de aprendizaje:
1. **Sopa de Letras** (Juego 01)
2. **Pintura Mágica** (Juego 02 - con guardado de imagen en servidor)
3. **Trivia Interactiva** (Juego 03)

### 🚀 Características
- Diseño responsivo con **Bootstrap 5**.
- Sistema de guardado en archivos **JSON** (sin necesidad de base de datos SQL).
- Captura de Canvas de HTML5 para guardar dibujos como PNG.

### 🛠️ Instalación
1. Clona el repositorio.
2. Súbelo a un servidor con PHP 7.4 o superior.
3. Asegúrate que la carpeta `dibujos/` tenga permisos de escritura.
